- 4 bits for a register
- registers each hold 1 byte

|------|------|-----------------|

| 0001 | r1   | General use     |

| 0011 | r3   | General use     |
| 0100 | r4   | General use     |
| 0101 | r5   | General use     |
| 0110 | r6   | General use     |
| 0111 | r7   | General use     |
| 1000 | r8   | General use     |
| 1001 | r9   | General use     |
| 1010 | r10  | General use     |
| 1011 | r11  | General use     |
| 1100 | r12  | General use     |
| 1101 | r13  | General use     |

| 1111 | ra   | return address  |
